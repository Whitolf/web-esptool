[![license][license-img]][license-url] [![issues][issues-img]][issues-url] [![commits][commits-img]][commits-url]

Flash Espressif chips on your web browser!

## Development

```sh
npm install
npm run serve
```

## Features / TODOs

- [x] Auto reset
- [x] Stub loader
- [x] Compression
- [x] [UF2](https://github.com/microsoft/uf2) image format
